# Workplace C Task JSONs

This directory contains the complete downloaded NeuroGolf JSON files for the 67 primary tasks assigned to owner C.

Source: `E:/kagglegolf/data/raw/neurogolf-2026/taskNNN.json` copied from the Kaggle competition data package.

Open the C viewer through a local server from the repository root:

```powershell
python -m http.server 8771 --bind 127.0.0.1 --directory "E:\kongming\NGC-work\workplace C"
```

Then open `http://127.0.0.1:8771/task_viewer_C.html`.

The full 400-task mirror is in `../neurogolf_400_tasks/`.
